# VK-Stats-Dashboard
# 🏏 Virat Kohli International Matches Stats – Power BI Dashboard  

## 📌 Project Overview  
This project presents an **interactive Power BI dashboard** analyzing the international career stats of **Virat Kohli**.  
It highlights his **runs, match performances, opponent statistics, and ground-wise insights**, turning cricket data into actionable insights.  

---

- **Opponent’s Matches Covered:** 516

## 🛠 Tools & Technologies  
- **Power BI** → Data visualization & interactive dashboard  
- **DAX** → Custom measures & calculations  
- **Excel/CSV** → Dataset source  
- **Power Query** → Data cleaning & transformation  

---

## 📂 Dashboard Features  
✔️ Runs scored by top 5 grounds  
✔️ Maximum runs scored vs different opponents  
✔️ Yearly, quarterly & monthly breakdown of runs  
✔️ Key KPIs – Total Runs, Maximum Runs, Opponent Matches, Run Rate  
✔️ Interactive filters to analyze performance trends
